Comandos más usados en GIT

- Inicializar un repositorio --> git init
- Clonar un repo remoto a local --> git clone + url del repo
- Vincular un repo local con uno remoto --> git remote add origin + url del repo
- Para añadir todo los cambios a stage --> git add .
- Para añadir un solo archivo/directorio a stage --> git add + nombre
- Para hacer un commit con el mensaje --> git commit -m "mensaje"
- Para subir los cambios al remo remoto --> git push origin + nombreDeLaRama
- Para traer los cambios de un repo remoto --> git pull origin + nombreDeLaRama
- Para moverme de una rama a otra, o hacia un commit --> git checkout + nombreDeLaRama/hashDelCommit
- Para crear una rama --> git branch + nombreDeLaRama
- Para ver el historial de commits --> git log
